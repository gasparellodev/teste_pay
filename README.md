# Documentação Front End

# Requisitos
Ter instalado o nodejs na versão 20.0 ou superior. <br /> 
Instale as dependências: <b> npm install </b> <br />
Rodar o projeto: <b> npm run dev </b> <br />
O projeto irá rodar na porta 8000, portanto acesse: <b> http://localhost:8000/login </b>
